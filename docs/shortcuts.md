# Shortcuts

## Applications

Super + T → Terminal

Super + W → Browser

Super + E → File Manager

## Caelestia

Insert → Launcher

Super + D → Dashboard

Super + N → Sidebar

Super + Esc → Session Menu

## Clipboard

Super + V → Clipboard History

## Magic Workspace

Super + S → Toggle

Super + Shift + S → Move Window

3 Finger Swipe Up → Toggle

3 Finger Swipe Down → Toggle

## Screenshots

Print → Fullscreen Screenshot

Super + Print → Region Screenshot

Super + Shift + Print → Save Screenshot

## Window Management

Super + Q → Close Window

Super + F → Fullscreen

Super + Shift + F → Floating Window
