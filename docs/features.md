# Features

## Desktop Environment

* Hyprland window manager
* Caelestia shell integration
* Arch (Linux) base system

## Customizations

* Gruvbox-inspired color scheme
* Custom window border colors
* Reduced window gaps
* Dashboard customizations

## Productivity

* Magic workspace (scratchpad)
* Rofi clipboard manager
* Fish shell
* Foot terminal

## Gestures

* Three-finger swipe up → Toggle magic workspace
* Three-finger swipe down → Toggle magic workspace

## Screenshots

* Fullscreen screenshot to clipboard
* Region screenshot to clipboard
* Region screenshot saved to file

## Utilities

* Clipboard history with cliphist + rofi
* Audio controls
* Brightness controls
* Media controls
* Workspace shortcuts
